    Instagram, but make it out of this world! Built in React, Spacestagram uses NASA's Astronomy Picture Of The Day API to render pictures of the day for the last 20 days and saves your likes in local storage.

  • Shopper Mapper

    A React app that uses multiple MapQuest API endpoints to get the users location and return a list of suggested places to go within a 10km radius, based on the users selected category of place. Choose from coffee shops, restaurants, bars, grocery stores and more.

    This app uses the Open Weather API to get the weather based on the users city. Depending on the weather in the users current location, the app returns a craft beer suggestion from an array of 30+ beers from local Toronto breweries.
